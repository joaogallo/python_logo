# PythonLogoBR

O projeto **PythonLogoBR** é uma iniciativa para traduzir e adaptar o módulo `turtle` do Python para o português brasileiro, tornando-o mais acessível para o ensino de programação e lógica de programação para crianças no Brasil.

## Sobre o Projeto

O módulo `turtle` é uma ferramenta popular no ensino inicial de programação, oferecendo uma forma visual e interativa de aprender conceitos básicos. No entanto, a barreira do idioma pode ser um desafio para crianças que estão começando a programar em países de língua portuguesa. O PythonLogo visa superar essa barreira, proporcionando uma experiência de aprendizado mais inclusiva e acessível.

## Funcionalidades

- Tradução completa das funções do módulo `turtle` para português.
- Documentação e exemplos em português.
- Recursos adicionais para professores e educadores.

## Como Usar

Para começar a usar o PythonLogo, siga os passos abaixo:

1. **Instalação:**
   ```bash
   pip install pythonlogo
   ```

2. **Utilização em um Programa Python:**
   ```python
   from pythonlogo import Tartaruga

   t = Tartaruga()
   t.para_frente(100)
   ```
